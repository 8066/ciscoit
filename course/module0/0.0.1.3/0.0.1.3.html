如何教学

电子实操这一设计理念充分运用了实践出真知的原则。课程包含极具互动性的嵌入式电子实操练习，可以帮助促进学习，获取更多知识，并使整个学习体验更加丰富，从而使课程内容更易理解。

互动练习

互动练习嵌入在各个章节中，可用于检查对知识的理解程度。其中一些练习使用“拖放”方式来完成（图 1）。另外一些练习使用“复选框”方法来完成（图 2）。您可以即刻查看分数。您还可以重置练习，想做多少遍都可以。

实验

实验是指专门在物理设备上进行的练习（图 3）。某些实验为课堂练习或小组练习，未必会用到设备。通过使用本课程中的实验来学习和实践实际技能，对于准备自己的 IT 职业生涯非常重要。在物理设备上进行操作不仅可以培养技能，还可以强化知识并树立自信。当您进入 IT 领域时，这一点非常重要。

Packet Tracer

在“网络”一章中，您将使用 Packet Tracer（Networking Academy 的网络模拟工具）（图 4）配置一个简单的网络。单击此处下载 Packet Tracer 的最新版本。Packet Tracer 初看上去可能很复杂，但它易于使用，而且有许多教程可以帮助您学习它的用法。Packet Tracer 是一款非常强大的工具，当您学习更多高级网络课程时会经常用到它。

Cisco Aspire

Cisco Aspire 是供单人独立完成的战略模拟游戏（图 5）。Networking Academy Edition 包含 IT 雇主期望求职者所具备的业务技能和沟通技能。单击此处了解有关 Cisco Aspire Networking Academy Edition 的详细信息。

评估

本课程中的每一个章节都包含在线测验和在线章节考试（图 6）。这些测验或考试是自动评分的，体现了您在哪些方面表现突出，哪些方面还需多加学习或实践。既有在线的课程结束终考，也有您将在课堂上完成的基于成绩的技能评估。
