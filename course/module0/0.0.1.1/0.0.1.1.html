欢迎学习信息技术 (IT) 基础课程。本课程的目的是向您介绍计算机硬件和软件，以及操作系统、网络连接概念、移动设备、IT 安全和故障排除。这些在线课程材料将帮助您掌握在 IT 领域中担任技术人员职位所需的必备技能。每章的开头将简介该章涵盖的具体技能。

您可以使用智能手机、平板电脑、笔记本电脑或台式机访问课程，与您的教师一起参加讨论，查看您的成绩，阅读或查看文本并使用交互式媒体测试您的理解程度。您还可以在移动设备上参加测验和考试。下载 Cisco Packet Tracer Mobile，完成整个课程中的 Packet Tracer 练习。

注意：即使您已下载了 Cisco Packet Tracer Mobile，使用 Packet Tracer 的测验和考试项目也无法在您的移动设备上正常运行。

单击此处访问 Cisco NetSpace 快速入门指南。本文档为您展示如何在 NetSpace 中进行浏览以及如何在课程中进行导航。它还解释如何访问您的作业和考试。
