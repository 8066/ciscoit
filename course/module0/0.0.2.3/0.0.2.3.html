现场技术人员、远程技术人员以及维修中心技术人员

您从事过计算机方面的工作，还获得了技术认证，这些经验可以帮助您胜任现场技术人员、远程技术人员或维修中心技术人员的工作。

计算机技术人员的职业不同，工作环境不同，只不过不同职业所需的各项具体技能水平会有所差异。在接受计算机技术人员培训时，您将需要培养以下技能：

  • 组装和升级计算机

  • 进行硬件和操作系统安装

  • 安装、配置和优化软件

  • 进行预防性维护

  • 排除计算机故障以及维修计算机

  • 与客户清楚沟通

  • 记录客户反馈的问题以及寻找解决办法所涉及的步骤

如图 1 所示，现场技术人员的工作场所各式各样，包括民宅、企业和学校。您可能为某家公司工作，只为该公司的计算机和网络系统提供支持服务。您也可能供职于向各种公司和客户提供计算机设备上门维修服务的公司。不管是哪种情况，都需要具备出色的故障排除技能和客户服务技能，因为您经常要和客户接触，处理很多硬件和软件问题。

如果您是一名远程技术人员，您可能在帮助台工作，回答遇到计算机问题的客户的来电或电子邮件，如图 2 所示。您创建工单并与客户沟通，以尝试诊断并修复问题。良好的沟通技能很重要，因为必须要让客户清楚地理解您的问题和说明。有些帮助台使用软件直接连接到客户的计算机来解决问题。作为一名远程技术人员，您可能就职于某企业，是其技术人员团队的一员，也可能在家工作。

作为一名维修中心技术人员，您通常不直接面对客户。如图 3 所示，维修中心技术人员通常在中央仓库或工厂执行计算机保修服务。
