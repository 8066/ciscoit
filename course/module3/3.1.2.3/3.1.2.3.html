安装内存

当计算机运行时，内存为 CPU 提供快速、临时的数据存储。内存是易失性存储器，也就是说，每次关闭计算机电源时其内容都会丢失。

您可以先在主板上安装内存，然后再将主板安装到计算机机箱中。在安装之前，请查阅主板制造商的主板文档或网站，以确保内存与主板兼容。

与 CPU 一样，内存也非常容易受 ESD 的影响。因此，在安装和拆卸内存时，请务必在防静电垫上操作，并戴上腕带或防静电手套。

要在主板上安装内存，请执行以下步骤：

第 1 步： （图 1）打开 DIMM 插槽上的锁定钮。

第 2 步： （图 2）将内存上的槽口与插槽上的突起对齐，然后用力向下按内存。

第 3 步： （图 3）确保锁定钮卡入到位。

警告：如果没有对齐，当计算机启动时，内存可能会损坏，并且可能严重损坏主板。

第 4 步： 确保每个内存模块完全插入插槽，并用锁定钮固定内存。

第 5 步： 目视检查是否有触点暴露在外。

安装提示：将内存正确按入插槽中时，会听到咔哒声并感觉到锁定钮卡入到位。
