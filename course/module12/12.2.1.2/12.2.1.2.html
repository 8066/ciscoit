访问 Windows 本地安全策略

在使用 Windows 计算机的大多数网络中，已在 Windows Server 上使用域配置了 Active Directory。Windows 计算机是域的成员。管理员配置适用于所有加入该域的计算机的域安全策略。用户登录 Windows 时，将自动设置帐户策略。

Windows 本地安全策略可用于不属于 Active Directory 域的独立计算机。要在 Windows 7 和 Vista 中访问本地安全策略，请使用以下路径：

开始>控制面板>管理工具>本地安全策略

在 Windows 8 和 8.1 中，请使用以下路径：

搜索>secpol.msc，然后单击 secpol。

如图所示，本地安全策略工具打开。

注意：在 Windows 的所有版本中，您可以使用“运行”命令 secpol.msc 打开本地安全策略工具。
