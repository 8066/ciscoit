IP 地址

现在，计算机通常都有两个 IP 地址版本。在 20 世纪 90 年代初，IPv4 网络地址几近用尽的问题令人担忧。Internet 工程任务组 (IETF) 开始寻找替代方案。这一行动拉开了制定现今 IP 版本 6 (IPv6) 的序幕。目前 IPv6 与 IPv4 并行存在，并已开始取代 IPv4。IPv4 地址长度为 32 位，用点分十进制数表示。IPv6 地址长度为 128 位，用十六进制数表示。图 1 显示了 IPv4 和 IPv6 地址示例。

IP 地址由网络管理员根据网络内的位置进行分配。当一台设备从一个网络移动到另一网络时，其 IP 地址最有可能发生更改。图 2 显示了拥有两个 LAN 的拓扑结构。该拓扑结构证实了在设备移动时 MAC 地址不会改变。但 IP 地址确实会改变。笔记本电脑已移动到 LAN 2 中。注意，笔记本电脑的 MAC 地址没有改变，但其 IP 地址的确发生了变化。

图 3 显示了在笔记本电脑上执行命令 ipconfig /all 的输出。输出显示 MAC 地址和两个 IP 地址。

注意：Windows 操作系统将网卡称为以太网适配器，将 MAC 地址称为物理地址。
