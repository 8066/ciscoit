最低硬件要求和操作系统平台兼容性

要想正确安装并运行操作系统，必须满足操作系统的最低硬件要求。

确定客户当前拥有的设备。如果需要进行硬件升级才能满足操作系统的最低要求，应进行成本分析以确定最佳的行动方案。有时，客户购买一台新计算机可能要比升级当前系统的成本更低。在其他情况下，升级一个或多个以下组件可能更为经济高效：

  • 内存

  • 硬盘驱动器

  • CPU

  • 显卡

  • 主板

注意：如果应用所需的硬件要求超过操作系统的硬件要求，则必须满足应用的其他要求才能让应用正常运行。

确定最低硬件要求后，请确保计算机中的所有硬件与您为客户选择的操作系统兼容。

Microsoft 兼容中心

Microsoft 提供一个在线 Windows 兼容中心，可让技术人员检查软件和硬件的兼容性，如图所示。该工具提供了一个详细的硬件清单，这些硬件经过测试和证明可与 Windows 配合使用。如果客户的任何现有硬件不在此列表中，可能需要升级这些组件。

单击此处转到 Microsoft 兼容中心。

要在结果页面上查看 Windows 兼容性状态，请单击产品以显示产品的详细信息页面。使用“兼容性状态”旁边的下拉菜单可以选取和查看与您最相关的 Windows 版本的兼容性状态。
