还原、刷新和恢复

系统还原

此工具可将计算机还原到早期的还原点状态。由于可从 Windows 内或在启动时访问系统还原工具，所以此工具可帮助您修复已损坏的系统，使其重新启动。图 1 显示了 Windows 8.x 中的系统还原工具。图 2 显示了 Windows 7 和 Windows Vista 中的系统还原工具。

刷新电脑（仅限 Windows 8.x）

如图 3 所示，此工具可将计算机系统软件还原到出厂状态，而不删除用户文件或删除现代应用。随着时间的推移，计算机往往会变得更慢。这是因为文件系统的正常使用会产生大量片段文件、孤立的驱动程序和库。此工具可以使系统回到出厂状态，且不会太影响用户数据。刷新计算机将删除所有已安装的桌面应用。

恢复分区

装有 Windows 的某些计算机包含一个用户不可访问的磁盘区域。此分区称为恢复分区，其中包含的映像可用于将计算机恢复到原始配置。

恢复分区通常是隐藏的，以防恢复之外的其他用途。要使用恢复分区还原计算机，必须在计算机启动时使用特定的按键或按键组合。有时，从出厂恢复分区进行还原的选项位于 BIOS 中。请与计算机制造商联系，了解如何访问该分区和还原计算机的原始配置。

注意：如果操作系统因硬盘驱动器故障而损坏，恢复分区也可能也损坏了，并将无法恢复操作系统。
