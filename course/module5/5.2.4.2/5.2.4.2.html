磁盘管理工具

多重启动设置需要多个硬盘驱动器或具有多个分区的硬盘驱动器。要创建新的分区，请访问磁盘管理工具。Windows 8.x（图 1）、Windows 7 和 Windows Vista（图 2）均提供了磁盘管理工具。您还可以使用磁盘管理工具完成以下任务：

  • 查看驱动器状态

  • 扩展分区

  • 拆分分区

  • 分配驱动器号

  • 添加驱动器

  • 添加阵列

要访问 Windows 8.x 中的磁盘管理工具，请访问“开始”屏幕并键入 diskmgmt.msc。单击搜索结果列表中显示的“磁盘管理工具”图标。

要访问 Windows 7 和 Windows Vista 中的磁盘管理工具，请使用以下路径：

开始 > 右键单击计算机 > 管理 > 选择磁盘管理

驱动器状态

磁盘管理工具显示每个磁盘的状态，如图所示。计算机中的驱动器显示以下状态之一：

  • 外部 - 从运行 Windows 的另一台计算机移至本计算机的动态磁盘

  • 状态良好 - 正常运行的卷

  • 正在初始化 - 正在转换为动态磁盘的基本磁盘

  • 丢失 - 已损坏、已关闭或已断开的动态磁盘

  • 未初始化 - 不包含有效签名的磁盘

  • 联机 - 可以访问且未显示任何问题的基本或动态磁盘

  • 联机（错误） - 动态磁盘上检测到 I/O 错误

  • 离线 - 已损坏或不可用的动态磁盘

  • 不可读取 - 经历硬件故障、损坏或 I/O 错误的基本或动态磁盘

使用除硬盘驱动器以外的其他驱动器时，例如光驱中的音频 CD 或空的可移动驱动器，可能会显示其他驱动器状态指示符。
